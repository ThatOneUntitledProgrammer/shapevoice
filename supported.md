# ShapeVoice Supported Platforms

ShapeVoice is cross-platform and can run on any system that supports Python.

## Supported Operating Systems:
- Windows
- Linux
- macOS

---

## Requirements:
To run ShapeVoice, you must have:

- Python 3.x installed
- Required Python packages installed:
  - numpy
  - scipy

---

## Notes:
If the requirements are met, ShapeVoice should run without modification on all supported operating systems.
